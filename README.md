# Note App

A simple notes application built in React JS. The application was created while learning the basics of React JS, server-side queries (axios + json-server) and the scss preprocessor. It is obvious that there are a lot of suboptimal code parts in the application, but they will not be fixed because there is no sense in it.

Приложение было создано в процессе изучения базовых принципов работы React JS, запросов к серверу (axios + json-server), а также препроцессора scss. Очевидно, что в приложении очень много неоптимальных частей кода, однако они не будут исправлены тк в этом нет смысла
